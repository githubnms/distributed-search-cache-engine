## Scalable Distributed Cache System

A distributed caching system designed to simulate large-scale
cache architectures used in modern cloud platforms.

## Features
- Consistent hashing based request distribution
- Low latency cache operations
- Horizontal scaling simulation
- REST API for cache read/write

## Tech Stack
Python, FastAPI, Docker, Distributed Systems
